# Simulator Control Panel

A simple, professional web interface to control your simulator API routes.

## Features

- Start/Stop all simulator routes
- Real-time status monitoring
- Auto-refresh every 3 seconds
- Clean, modern UI
- Responsive design

## Usage

1. Make sure your simulator API is running on `http://localhost:8000`
2. Open `index.html` in your web browser
3. Click the buttons to start or stop different simulations

## Available Controls

- **Random Radar** - Generate random radar points
- **Radar Decoy** - Generate radar decoy points  
- **Main Simulation** - Run simulated flights
- **Drone Attack** - Trigger drone attack simulation
- **Rocket Attack** - Trigger rocket attack simulation

## Configuration

You can change the API URL in the input field at the top if your API is running on a different address.
